# o-O — 협업형 마인드맵 서비스
> **“생각을 말하면 → 자동으로 확장되고 → 하루 단위로 쌓이며 → 전체 아이디어 지도로 연결된다.”**
> 
> 
> 
> **“당신의 생각이, 세상의 흐름이 됩니다.”**
> 
> **— AI가 연결하는 생각의 구슬, o-O**
> 

## 기획배경

- 기존 마인드맵 툴은 모든 가지를 사용자가 직접 입력해야 하며, 아이디어가 많아질수록 관리, 구조화 어려움
- 사용자는 한순간 떠오른 **아이디어를 바로 기록**하고 싶지만, 구조화나 분류 작업이 번거로워 아이디어를 **여러 곳에 저장**하거나 **잊어버리는** 경우가 많음
- 비슷한 주제에 관심 있는 사람들끼리 의견을 확장·공유할 수 있는 공간 부재
- GPT 같은 AI를 사용하더라도 결과물이 흐름 없이 나열되기 때문에, 아이디어의 구조나 관계를 한눈에 파악하기 어렵다.
- 따라서 GPT와 LLM 기술을 활용해
    - ① 사용자의 생각을 자동으로 확장하고,
    - ② 비슷한 생각을 가진 사람들의 아이디어를 연결하며,
    - ③ 사회 전체의 아이디어 흐름을 시각화할 수 있는 지능형 마인드맵 플랫폼이 필요하다.

## 타겟

### 1) 개인 사용자

- 아이디어를 빠르게 구조화하고 싶은 **기획자, 학생, 창업 준비자**
- 콘텐츠 제작자를 위한 주 발굴 및 확장 도구
- 연구자, 논문 작성자, 발표 준비자

### 2) 팀/조직 사용자

- 기획 회의, 브레인스토밍, 연구 모임 등에서 실시간 협업 필요
- 음성 회의를 AI가 자동으로 정리하고, 회의록이 곧 마인드맵으로 생성되길 원하는 팀
- 아이디어 구체화가 필요한 사람 : **기획자, 학생 등**

## 주요 기능

1. **음성 기반 아이디어 기록 (개인)**
    - 사용자가 앱에서 음성으로 아이디어 말하면, STT 통해 텍스트로 변환
    - AI가 내용을 분석하여 자동으로 핵심 키워드를 추출하고 마인드맵을 생성
2. **달력형 타임라인 (나의 아이디어 히스토리)**
    - 하루 단위로 내가 생성한 키워드 / 마인드맵 확인
3. **개인/팀 마인드맵**
    - 키워드 클릭 시 **새로운 키워드 추천 및 추가/수정/삭제 가능**
        - 추가:  선택한 키워드 하위로 새 키워드 연결
        - 삭제
            - 자식 노드 전체 삭제
            - 자식 노드 분리 → 분리된 노드를 클릭하면 AI가 자동으로 새로운 연결 생성
    - **공개 범위 선택**: private/public 선택 가능
        - public 설정 시 전체 탐색 맵에 자동 반영 
        ( 한 번 public 한 마인드맵은 전체 탐색 맵에 반영되며, private 으로 돌려도 반영 그대로 )
    - **AI 자동 병합** : 유사한 아이디어를 병합·요약해 중복 제거
    - **추천 기능**
        - 클릭한 키워드와 키워드에 해당하는 내용을 바탕으로 새로운 키워드 2~3개 추천
            - 전체 마인드맵에서 연결되어있는 키워드 중 자식 많이 달려있는 키워드로 추천
            - AI 추천 키워드
    - **참고 자료 자동 첨부**:
        - AI의 피드백에 관련 논문·기사 링크 제공
        - 출처 제공
    - **Markdown 형식의 기획안 제작**
        - 확장된 마인드맵을 기획서 형태로 정리
    - **팀 협업 기능 :**
        - 팀장이 아이디어 먼저 입력한 이후, 팀원이 각자 아이디어 입력
        - AI가 의미상 유사한 가지로 자동 정리
        - 실시간 동시 편집 및 커서 표시를 지원하며, 팀 음성 채팅 중 발화 내용도 실시간으로 마인드맵 반영된다.
4. **전체 마인드맵 (탐색형)**
    - 모든 public 아이디어가 모여 자동으로 생성되는 **“전체 생각 지도”**
    - 수정은 불가하며, 탐색/검색 중심의 시각화 기능 제공
    - AI 가 중복된 아이디어를 병합해 클러스터 단위로 그룹화
    - 키워드 검색 시 :
        - 존재하는 키워드 → 해당 키워드를 중심으로 마인드맵 시각화
        - 존재하지 않는 키워드 → 의미상/글자상 유사한 키워드를 트렌드 형식으로 제시
5. **트렌드 보드**
    - Public 아이디어를 주간 단위로 분석해
        - 가장 많이 언급된 주제
        - 급상승 키워드 / 주요 이슈
        
        를 시각화하여 제공
        
    - 주제 클릭 시 해당 주제를 중심으로 마인드맵 탐색 가능

## 사용자 여정 (User Flow)

### 1. 홈 및 초기 진입 (Home & Notification)

- 사용자는 Google 계정으로 로그인한다.
- 홈 화면에서 최근 접근한 마인드맵을 카드 형태로 확인할 수 있다. (모바일)
- AI가 트렌드 키워드와 브레인스토밍 주제를 추천한다.
- 사용자는 브레인스토밍을 시작할 아이디어를 입력(텍스트/음성)한다. (웹:텍스트/모바일:음성)
- 일정 시간 미작성 시 리마인더 알림을 통해 아이디어 작성을 유도한다.

→ 결과**:** 사용자가 입력을 시작하고 첫 번째 아이디어를 캡처함.

---

### 2. 아이디어 입력 및 자동 마인드맵 생성 (Mind Map 생성)

- 음성 입력 시 STT 변환으로 텍스트 자동 생성.
- 입력된 텍스트/이미지/영상 링크를 기반으로 AI가 핵심 키워드를 분석/생성한다.
- 사용자는 제안된 키워드를 선택하거나 클릭한 키워드에 대해 확장을 요청할 수 있다.
- 공개 범위(Public / Private)를 설정할 수 있다.
- AI가 유사 아이디어를 자동 병합·요약하여 중복 제거.

→ 결과: 개인 마인드맵의 초안이 생성되고, 핵심 가지들이 구성됨

---

### 3. 마인드맵 편집 및 시각화 (Mind Map 조작)

- 사용자는 노드를 클릭하여 하위 키워드를 추가·수정·삭제한다.
- 노드 삭제 시 자식 노드를 분리하거나 자식 노드까지 삭제를 선택할 수 있으며, 
 자식노드 분리 선택 시 AI가 새로운 연결 제안.
- 각 노드는 색상 변경과 구슬 크기로 시각적으로 구분된다.
- AI는 참고 자료(논문·기사)와 출처를 함께 제공한다.

→ 결과: 사용자가 시각적으로 구조화된 마인드맵을 완성하고 관리.

---

### 4. 개인 히스토리 및 마이페이지 (MyPage)

- 하루 단위로 생성된 아이디어가 자동 저장되고, 마이페이지에서 달력 UI를 통해 날짜별 기록 여부 확인.
- 클릭 시 구슬 뷰로 이동하여 해당 날짜의 아이디어를 다시 볼 수 있음.
- 오늘 생성한 키워드 목록을 확인 가능.
- 프로필 이미지 및 닉네임 수정 가능.
- 개인 대시보드에서 마인드맵 리스트(공개 여부, 생성일, 주제 등) 확인.

→ 결과: 개인의 사고 흐름을 시간 축으로 관리하고 성장을 시각화함.

---

### 5. 트렌드 탐색 및 복제 생성 (Trend)

- Public 아이디어를 기반으로 주간 인기 키워드 및 트렌드 확인.
- 특정 키워드 검색 및 키워드 중심 마인드맵 열람.
- 마음에 드는 마인드맵 또는 키워드를 복제 생성하여 개인 맵으로 가져오기.

→ 결과: 다른 사용자의 생각을 탐색하며, 사회적 지식 흐름 속에서 아이디어를 확장.

---

### 6. 협업 및 기획안 제작 (Collaboration)

- 사용자(팀장)는 팀원을 초대하고 아이디어를 먼저 입력한 후, 협업 모드로 진입.
- 팀원들과 실시간으로 공동 편집하며, 커서 위치 확인 가능.
- 음성 회의에서 실시간으로 마인드맵 키워드 추가 확인.
- 협업 중 AI가 입력된 아이디어를 자동 병합·정리.
- 팀원 권한(편집/읽기/초대)을 관리할 수 있다.
- 완성된 마인드맵을 기획안 형태로 변환할 수 있다.

→ 결과: 팀 단위 아이디어 통합 및 문서화 프로세스 완성.

## 사용 기술

### 🧠 AI / NLP

- **Llama / OpenAI API / 로컬 LLM 연동** : 아이디어 생성, 요약, 병합, 트렌드 분석
- **Whisper / Web Speech API** : 음성 입력 → 텍스트 변환(STT)
- **Sentence-BERT + ElasticSearch** : 의미 유사도 분석, 아이디어 클러스터링
- **HuggingFace Transformers** : 문맥 파악, 감정 분석, 카테고리 분류

---

### 🖥 Backend

- **Java + Spring Boot** : 핵심 비즈니스 로직, 사용자 인증, 마인드맵 CRUD API
- **MSA 구조 기반 서비스 분리**
    - User Service / Idea Service / Collaboration Service / Notification Service 등 독립 배포
- **Spring Cloud Gateway + OpenFeign** : API 라우팅 및 서비스 간 통신
- **Redis** : 세션 관리, WebSocket 연결, 협업 상태 캐싱
- **RabbitMQ 또는 Kafka** : 실시간 이벤트 전달 및 AI 처리 로그 수집

---

### 🗂 데이터베이스

- **PostgreSQL (RDB)** : 사용자 정보, 마인드맵 메타데이터, 협업 기록 저장
- **MongoDB (NoSQL)** : 노드-엣지 구조의 마인드맵 데이터를 JSON 형태로 저장
- **Redis** : 임시 데이터, 실시간 편집 세션, 트렌드 통계 캐싱
- **ElasticSearch** : 전체 마인드맵 탐색, 키워드 기반 검색 및 트렌드 분석

---

### 💡 Frontend (Web / App 통합)

- **React + TypeScript** : 메인 웹 클라이언트 (SPA, PWA 대응)
- **D3.js / Cytoscape.js** : 마인드맵 및 네트워크 구조 시각화
- **Recharts** : 트렌드 분석 및 사용자 통계 시각화
- **React-Calendar, FullCalendar** : 캘린더형 히스토리 UI
- **Flutter** : 모바일 앱 전용 UI 최적화 및 동일 UX 제공

---

### ☁️ Infra / DevOps

- **AWS EC2 + S3** : 서비스 운영 및 마인드맵 미리보기 저장
- **Kubernetes + Docker** : 컨테이너 기반 오케스트레이션 및 자동 확장
- **Jenkins / GitLab CI/CD** : 코드 배포 자동화 및 Blue-Green Deployment 적용
- **Prometheus + Grafana + Loki** : 로그 수집, 모니터링, 장애 대응

---

### 🔔 인증 및 메시징

- **Firebase Auth** : 구글 계정 기반 사용자 인증
- **Firebase Cloud Messaging (FCM)** : 푸시 알림 및 리마인더 발송
- **WebSocket + Redis** : 실시간 협업 상태 동기화
